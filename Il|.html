Heres a compiler to a coding language that only uses I, l and |


The code is based on Js. All built in functions come with ( already. For ifs, elses and functions use line breaks for the open curly bracket.
Table 1:
0) ||| is new line (or ";" in Js and is absolutely mandatory)
1) ||I is bitwise not, !
2) ||l is a print command ("console.log(" in Js)
3) |I| is define variable ("var" in Js, is already followed by a meta character)
4) |II is =, used for variable definitions.
5) |Il is ), used to close parentheses.
6) |l| is (, used for functions you create.
7) |lI is ", used for strings.
8) |ll is |, used for or
9) I|| is &, used for and or for strings with values inside, (js, $"Your score is {x}")
10) I|I is ^, used for XOR 11) I|l is making a bitwise operation a boolean opperation. 12) II| is if, ("if(" in js) 13) III is else, ("else(" in js)
