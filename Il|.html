Heres a compiler to a coding language that only uses I, l and |


The code is based on Js. All built in functions come with ( already.
Table 1:
0) ||| is new line (or ";" in Js and is absolutely mandatory)
1) ||I is a meta character, look at the next table for more. 2) ||l is a print command ("console.log(" in Js) 3) |I| is define variable ("var" in Js, is already followed by a meta character) 4) |II is =, used for variable definitions. 5) |Il is ), used to close parentheses. 6) |l| is (, used for functions you create.
